

(1) follow guide to create partitions. 

(2) make configs. 

(3) copy installing-essential.nix into the configuration. Then run the install process. 

(4) upon reboot, clone repository into home folder. 

(5) make the following edits to the config:
    (a) copy hardware config into host file. 
    (b) copy stateVersion into host file. 
    (c) make sure the correct hostname is in the host file. 

(6) run "sudo nixos-rebuild switch --flake ~/my-configs/#<hostname>"
